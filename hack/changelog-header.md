### Linux

```shell
curl -L https://github.com/jenkins-x-plugins/jx-charter/releases/download/v{{.Version}}/jx-charter-linux-amd64.tar.gz | tar xzv 
sudo mv jx-charter /usr/local/bin
```

### macOS

```shell
curl -L  https://github.com/jenkins-x-plugins/jx-charter/releases/download/v{{.Version}}/jx-charter-darwin-amd64.tar.gz | tar xzv
sudo mv jx-charter /usr/local/bin
```
